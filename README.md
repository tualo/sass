# Tualo Office - SASS

Eine PHP-Bibliothek für die Integration und Kompilierung von SASS/SCSS in Tualo Office.

## Beschreibung

Dieses Paket bietet SASS/SCSS-Kompilierung und CSS-Minifizierung für Tualo Office Anwendungen. Es ermöglicht die Verwendung moderner CSS-Präprozessoren in Ihrer Tualo Office Installation.

## Installation

```bash
composer require tualo/sass
```

## Funktionen

- SASS/SCSS Kompilierung
- CSS Minifizierung
- Integration in Tualo Office
- CLI-Kommandos zur Verwaltung
- Automatische Route-Registrierung

## Anforderungen

- PHP >= 7.4
- Tualo Office Framework
- matthiasmullie/minify
- sass compiler (zum Beispiel npm i -g sass)
